***LINUX***



* History -c - limpiar historial en terminal
* cd / cd .. - entrar o salir de carpetas
* mkdir - crear carpeta
* put Nul - Crear un archivo
* dir - listar carpetas
* ls - listar carpetas/archivos
* put Nul - crear archivo
* get - descargar archivo
* dh client enp0s9 - Despertar puerto
* sha256sum apache2\_2.4.deb > apache2\_2.4.deb.sha256 # Entra a donde están tus instaladores, actualiza el sha256
* &#x20;cd /mnt/practicas - entrar a carpeta de mis practicas
* openssl x509 -in /etc/ssl/reprobados/servidor.crt -text -noout | grep -i "Subject:" - ver credenciales de vsftpd
